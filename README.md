# About

This project is based on the Tome project.

# Requirements

- PHP 7+
- [Composer](https://getcomposer.org/)
- [Drush](https://github.com/drush-ops/drush-launcher#installation---phar)
- SQLite and the related PHP extensions

## Usage

Visit https://tome.fyi/docs/hosting/github-pages/ for instructions for building.
